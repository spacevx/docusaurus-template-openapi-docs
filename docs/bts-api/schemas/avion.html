Avion

Représentation d'un avion dans un système de gestion de flotte

idinteger

Identifiant unique de l'avion

immatriculationstringrequired

Numéro d'immatriculation de l'avion

Example: F-ABCD
marquestringrequired

Constructeur de l'avion

Example: Airbus
modelstringrequired

Modèle spécifique de l'avion

Example: A320
fuelCapacitynumber

Capacité totale de carburant de l'avion

Example: 20000
fuelLevelnumber

Niveau actuel de carburant dans l'avion

Example: 15000
hangarIDintegerrequired

ID du hangar où l'avion est stocké

Possible values: >= 1

commissioningDatedate

Date de mise en service de l'avion

Example: 2020-01-15
lastRevisiondate

Date de la dernière révision de maintenance

Example: 2023-06-30
lastFlyingdate

Date du dernier vol

Example: 2024-01-10
lastPilotIDintegernullable

ID du dernier pilote ayant piloté l'avion

lastMaintenanceIDintegernullable

ID de la dernière opération de maintenance
